golang学习笔记

语言简介

golang 是 goole 公司开发的静态、编译型、跨平台的编程语言，其主要优点如下：

  1. 避免嵌套调用头文件，缩短编译时间；
  2. 生成 Native Code，缩短程序加载时间；
  3. GC 垃圾回收机制，减轻程序员负担。

变量

  • 所有值类型变量都会被隐式初始化

  • 简化声明只能在函数内部使用

  • 局部变量声明后必须被使用；接收某个函数返回值却不使用时应采用匿名变量_。

    v, _ = http.Get("https://www.baidu.com")

  • 解构变量声明允许部分变量被重复声明，比如下例中的变量b

    func main(){
	a, b := 1, 2
	fmt.Printf("a:\t%v-%p\nb:\t%v-%p\n", a, &a, b, &b)
	// a:1-0xc0000aa058
	// b:2-0xc0000aa070
	b, c := 3, 4
	// b:3-0xc0000aa070
	// c:4-0xc0000aa0a0
	fmt.Printf("b:\t%v-%p\nc:\t%v-%p\n", b, &b, c, &c)
}

常量

  • 常量的值必须在编译期确定，因此不能将函数返回值赋予常量。

    const a = math.Sqrt(4) // math.Sqrt(4) (value of type float64) is not constant

  • 批量声明常量时，未初始化的常量会继承上一个常量的初始值。

    const (
	n1 = 100
	n2
	n3
	n4 = 200
	n5
)

func main() {
	fmt.Println(n1, n2, n3, n4, n5) // 100 100 100 200 200
}

  • itoa自增变量初始值为0，后续未初始化常量会沿用前一个常量表达式，并且itoa自增1。

    package main

import "fmt"

const (
	Readable   = 1 << iota                        // 00000001
	Writable                                      // 00000010
	Executable                                    // 00000100
	RW         = Readable | Writable              // 00000011
	RWE        = Readable | Writable | Executable // 00000111
)

func main() {
	mask := 0b11
	if mask&RW == RW {
		fmt.Println("Readable & Writable")
	}
}

数据类型

  • 不支持任何隐式类型转换和提升，即使类型兼容也不行。甚至连原类型和别名类型相互转换都不行。

    x := 2
y := 1.1
// invalid operation: x + y (mismatched types int and float64)
x + y

  • byte 是 uint8 的别名，代表一个 ASCII 码的字符；runne 是 int32 的别名，代表一个 UTF-8 字符。

    var a byte = 200
var b uint8 = 200
fmt.Printf("a\t%b\n", a) // a	11001000
fmt.Printf("b\t%b\n", b) // b	11001000

  • golang 支持指针但不支持指针运算

    func main() {
	a := 1
	b := &a
	*b = 2
	fmt.Println(a) // 2

	b = b + 1 // cannot convert 1 (untyped int constant) to *int
}

运算符

  • 与 C 语言不同，自增、自减运算界定为语句而非表达式，因此该运算无返回值，且只有右增、右减。

    a := i++ // syntax error: unexpected ++ at end of statement
++i // syntax error: unexpected ++

条件语句

  • 区别与绝大多数编程语言，条件语句没有小括号；条件语句后还可写一条 statement 语句。

    if statement; condition {  
}

  • 即使只有一条语句，语句体的大括号也不能省略。

循环语句

  • 语言本身没有提供 while 循环，可用 for 循环实现。

    func main() {
	i := 0
    for i <= 10 { 
        fmt.Printf("%d ", i)
        i += 2
    }
}

switch语句

  • 某个 case 分支执行完毕后，默认跳过后续分支检测，除非在分支末尾使用 fallthrough关键字。

    switch num := 10 {
    case num < 50:
        fmt.Printf("%d is less than 50\n", num)
        fallthrough
    case num < 100:
        fmt.Printf("%d is less than 100\n", num)
        fallthrough
    case num < 200:
        fmt.Printf("%d is less than 200", num)
    }

  • 用无常量表达式的 switch 语句（相当于switch true）代替多重if-else语句

    num := 75
switch { 
case num >= 0 && num <= 50:
   fmt.Println("num is greater than 0 and less than 50")
case num >= 51 && num <= 100:
   fmt.Println("num is greater than 51 and less than 100")
case num >= 101:
   fmt.Println("num is greater than 100")
}

函数

  • 多返回值

    func main() {
	area, perimeter := rectangle(10, 5)
	fmt.Println("area", area) // area 50
	fmt.Println("perimeter", perimeter) // perimeter 30
}

func rectangle(length, width float64) (float64, float64) {
	area := length * width
	perimeter := 2*length + 2*width

	return area, perimeter
}

  • 可变参数在函数内部表示为切片，然而直接传递切片作为参数却是不允许的，应将切片展开后传递。

    func dispaly(message ...int) {
	for _, value := range message {
		fmt.Println(value)
	}
}

numbers := []int{1,2,3,4,5}
display(numbers) // 错误
display(numbers...) // 正确

进阶内容

数组

  1. 数组是值类型

    bosses := [...]string{"China", "Russia", "UK", "France", "USA"}
rogues := bosses
rogues[0] = "?"
fmt.Println(bosses) // [China Russia UK France USA]
fmt.Println(rogues) // [? Russia UK France USA]

  2. 使用字面量初始化数组时，用...操作符自动推断数组长度。

    // 原始写法
rgb := [3]string{"red","green","blue"}
// 省事写法
rgb := [...]string{"red","green","blue"}

  3. 使用range迭代数组时，会使用新建的迭代变量拷贝当前元素值，因此修改迭代变量不会对数组产生影响。

    nums := [...]int{1, 2, 3}
for _, v := range nums {
	v++
}
fmt.Println(nums) // [1 2 3]
for i, v := range nums {
	fmt.Printf("%p vs %p\n", &v, &nums[i])
}
// 0xc0000120b0 vs 0xc000014120
// 0xc0000120b0 vs 0xc000014128
// 0xc0000120b0 vs 0xc000014130

切片

  1. 切片是引用类型

    students := []string{"Tom", "Jerry", "Hans"}
men := students
men[0] = "Jason"
fmt.Println(students) // [Jason Jerry Hans]

  2. 切片的本质是结构体，底层引用了某数组中的元素；切片的大小成员代表切片的元素个数，容量成员代表被引用数组的元素个数。

    week := [...]string{"Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"}
  workday := week[0:4]
  workday[0] = "周一"
  fmt.Println(week) // [周一 Tuesday Wednesday Thursday Friday Saturday Sunday]
  fmt.Println("length:", len(workday), "capacity:", cap(workday)) // length: 4 capacity: 7

  3. 新增或删除元素都会在底层创建一个新的支撑数组。新增元素后，新数组的长度是切片原长度的两倍。

    committees := []string{"Tom", "Jerry", "Hans"} // 长度3
fmt.Printf("原始数组地址：%p\n", committees) // 原始数组地址：0xc000078480
committees[0] = "Washington"
fmt.Printf("修改元素后地址：%p\n", committees) // 修改元素后地址：0xc000078480
committees = append(committees, "jason") // 长度4
fmt.Printf("新增元素后地址：%p\n", committees)    // 新增元素后地址：0xc00004e060
fmt.Printf("新数组长度：%v\n", cap(committees)) // 新数组长度：6

字符串

  1. 字符串是值类型，编译器自动初始化栈空间为0，即空字符串。将字符串变量与nil进行比较会报错。

  2. 字符串的本质是 UTF-8 编码的 byte 类型数组，但是数组的字节数并不总等于数组的长度。

    slogan := "hello世界" // 5+2个元素
// len函数获取的是数组字节数，世界这两个汉字在UTF-8编码中共占用6个字节
fmt.Println("字符串字节数：", len(slogan))                   // 字符串字节数： 11
fmt.Println("字符串长度：", utf8.RuneCountInString(slogan)) // 字符串长度： 7

  3. rune类型代表一个 UTF-8 码点。迭代字符串的两种方法如下，推荐第二种方法。

    func foo(s string) {
	tmp := []rune(s) // 将字符串转换为rune切片
	for i := 0; i < len(tmp); i++ {
		fmt.Printf("%c", tmp[i])
	}
}

// 推荐使用
func bar(s string) {
	for _, v := range s {
		fmt.Printf("%c", v)
	}
}

包

  • 包可以更好地组织源代码并能避免命名冲突的问题。go 工作目录下的 src 目录存放了第三方和自定义包，一般一个文件夹对应一个包，根据惯例文件夹应与包同名（非强制要求）。使用 import 关键字导包时会自动搜索 src 目录。

  • 导出名称的首字母必须大小。

  • 可执行程序必须创建一个 main 包，并包含一个 main 函数，这是程序的入口点。

  • 每个包都可包含一个无参数、无返回值的 init 函数，用于执行包级变量初始化等操作。尽管一个包可能被间接包含多次，但 init 函数 只执行一次。
